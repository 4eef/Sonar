Experimental prototype of ultrasonic sonar with increased noise immunity (January-February 2016)

Русский:

Проект разрабатывался как экспериментальная часть дипломной работы магистра. Темой дипломной работы является повышение разрешающей способности по дальности ультразвуковых эхолокаторов БПЛА. Повышение разрешающей способности по дальности предполагается произвести применением сложных шумоподобных сигналов с последующей согласованной фильтрацией принятого сигнала автокорреляционной функцией.
Ультразвуковые эхолокаторы являются оптимальными для применения на беспилотных ЛА из-за всепогодности и низких требований к вычислительной мощности бортовой ЭВМ. Также, их разрешающая способность на частоте 40 кГц позволяет обнаруживать препятствия габаритами >4,125 мм. Это обусловливает выбор типа эхолокатора для изготовления прототипа.
За основу была взята схемотехника автомобильного парковочного ультразвукового радара. Она позволяет гибко управлять фазой передаваемого сигнала воздействием на первичную обмотку трансформатора. В приёмный тракт схемы был добавлен каскад усилителей для осуществления автоматической регулировки усиления принимаемого сигнала для компенсирования его затухания. Усиленный сигнал далее поступает на микрофонный вход и обрабатывается программным обеспечением на ПК.
Для экспериментального исследования, расширение спектра передаваемого сигнала осуществлено путём применения последовательности Баркера с N = 11. Последовательность периодически генерируется микроконтроллером ATtiny44A.
В результате экспериментального исследования было получено:
- математическая модель автомобильных пьезоэлектрических преобразователей для нахождения оптимального типа модуляции сигнала и определение его параметров посредством симуляции;
- увеличение рабочего радиуса ультразвуковых эхолокаторов на основе автомобильных пьезопреобразователей до 20 м;
- увеличение помехозащищенности ультразвуковых эхолокаторов применением последовательности Баркера (N = 11) и согласованной фильтрации;
- компенсирование затухания сигнала применением автоматической регулировки усиления сигнала.

English:

Project was developed as a hardware-software part for master's diploma. Main purpose of the project was to increase the resolution by distance of UAV's ultrasonic sonars by using complex noise-like signals and followed by matched filtration with autocorrelation function.
Ultrasonic sonars are optimal for use in UAV systems because of all-weather applicability and low requirements to computing power of onboard computer. Also its own resolution on 40 kHz frequency makes possible to recognize obstacles with dimensions from 4.125 mm and higher. This causes the selection of sonar type for prototype development.
Schematics is based on automotive ultrasonic parking sonar. This allows the flexibility in signal's phase control by means of impact on primary winding of transformer. To compensate received signal attenuation automatic gain control circuit was added to reception path of the scheme. Amplified signal is supplied to microphone input and proccessed by software on PC (not included to repository).
For experimental research, Barker code with N = 11 was used to broaden the transmitted signal spectrum. Signal is periodically generated by AVR ATtiny44A microcontroller.
As a result of an experimental research was obtained:
- automotive piezoelectric converter's mathematical model for purposes of optimum signal modulation method and parameters determination by simulation;
- increasing the working range of ultrasonic echolocators based on automotive piezoelectric converters to 20 m;
- increasing the noise immunity of ultrasonic echolocators by using the Barker code (N = 11) coupled with correlation function;
- compensating the signal attenuation by using an automatic gain control.
